## Comprendre

[Husky](https://typicode.github.io/husky/) est un outil indispensable pour automatiser la qualité du code directement dans les hooks Git.

Avec Husky, tu peux exécuter des tâches comme ESLint, Prettier ou des tests avant chaque commit ou push. Cela garantit un code propre et stable tout au long du projet. Dans ce module, tu apprendras à l'installer, le configurer, et l'utiliser efficacement pour sécuriser ton workflow de développement.”

### Ajouter husky

```bash
npnm add husky --save-dev
```

Créer un dossier `.husky`

```bash
#!/bin/sh

echo "🧪 Running tests before commit..."

pnpm run test-husky

if [ $? -ne 0 ]; then
  echo "❌ Tests failed. commit aborted."
  exit 1
fi

echo "✅ Tests passed. Commit allowed."
```

```bash
#!/bin/sh

echo "🧪 Running tests before push..."
pnpm run test-husky
pnpm lint
pnpm build
if [ $? -ne 0 ]; then
  echo "❌ Tests failed. Push aborted."
  exit 1
fi

echo "✅ Tests passed. Push allowed."
```
